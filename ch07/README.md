## 练习 7.18

将类内部分成员设置为外部不可见，而提供部分接口给外面，这样的行为叫做封装。封装隐藏了具体的实现，当我们使用某个抽象数据类型时，只需要考虑它提供什么接口操作，而不用去考虑它的具体实现。

## 练习 7.19

`public`: 构造函数，`get_name`, `get_address`
`private`: `name`, `address`
数是暴露给外部的接口，因此要设为public；而数据则应该隐藏让外部不可见。
